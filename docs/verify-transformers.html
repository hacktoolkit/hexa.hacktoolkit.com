üö® Common Issues

Issue: No console messages appear

Solution: Make sure Console tab is open BEFORE switching providers

Issue: "Failed to fetch" or network errors

Causes:

  • Firewall blocking HuggingFace CDN
  • No internet connection
  • Corporate proxy blocking downloads

Solution: Check internet connection, try different network

Issue: Loading forever, no progress

Solution:

  • Check Network tab - are files downloading?
  • Refresh page and try again
  • Try different browser (Chrome/Edge recommended)

Issue: Model loads but generation fails

Solution: Check console for specific error, may need to restart browser
